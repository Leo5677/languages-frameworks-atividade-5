# languages-frameworks-atividade-5

## Construção:
1. BACKEND: FLASK
2. FRONTEND: HTML, CSS E JS

## Como Utilizar?
Após clonar o repositório, execute os comandos abaixo: 

1. pip install -r requirements.txt <br>
2. py -m venv env <br>
3. cd .\env\Scripts\ <br>
4. .\activate <br>
5. cd .. (Executar 2x) <br>
6. py app.py <br>
7. Abra no navegador: http://localhost:8080/
